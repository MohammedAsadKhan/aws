## About This Certification

The AWS Certified AI Practitioner validates foundational knowledge of AI, ML, and generative AI concepts and their implementation on AWS. It covers responsible AI, foundation models, prompt engineering, and AWS AI/ML services.

**Passing score:** 700/1000  
**Exam format:** 65 questions, 90 minutes  
**Question types:** Multiple choice, multiple response
