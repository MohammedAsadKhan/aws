## About This Certification

The AWS Certified Cloud Practitioner validates foundational cloud knowledge and AWS services. It covers cloud concepts, core AWS services, security, pricing, and support models.

**Passing score:** 700/1000  
**Exam format:** 65 questions, 90 minutes
